BFS & DFS Visualiser

A simple web app that visualizes how
Breadth-First Search (BFS) and Depth-First Search (DFS) explore graphs.

Built using:
	•	HTML
	•	CSS
	•	JavaScript

Features
	•	Visual animation of BFS and DFS
	•	Shows traversal order clearly
	•	Beginner-friendly interface

Notes

I used Gemini as a coding assistant while building this project.
It helped with debugging and structuring some parts of the code,
but I made sure to understand the logic and modify the implementation myself.
